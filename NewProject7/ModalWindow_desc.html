Ajax example: Modal Window (pop-up)

Description

This example shows how to use a modal window to add wishes at site. The wish list will be updated automatically without refresh of whole page.

Steps to recreate

  1. Add Update Panel component selecting it from the "Forms" tab of the Toolbox window. Leave the default Panel1 name.
  2. Run Grid Builder to create the Grid based on the wishes table inside the Panel1 panel. Delete fields 'wish_id' and 'wish_email' from field list.
  3. Add link control into botton of wishes grid, with Static Text 'Add New' and default name Link1.
  4. Add Dialog Panel component (outside Panel1) selecting it from the "Forms" tab of the Toolbox window. In dialog set Width and Height attributes as empty. Leave the default Panel2 name.
  5. Add Update Panel component (inside Panel2) selecting it from the "Forms" tab of the Toolbox window. Leave the default Panel3 name.
  6. Run Record Builder to create the Record based on the wishes table inside the Panel3 panel. In Action tab uncheck Allow Update and Allow Delete checkboxes,
  7. In HTML mode add id="Panel2Panel3wishes1Error" to any tag in Error block for wishes1 record.
  8. For wishes1 record, change type of wish_date textbox to hidden, set it properties:
    • Format: mm/dd/yyyy h:nn AM/PM
    • Default Value: CurrentDateTime
    • Caption: Wish Date
  9. For textbox wish_email in wishes1 record, add Validate Email action to On Validate event, set:
    • Control Name: wish_email
  10. Add Condition feature to Panel1 panel, set it attributes as:
    • Condition: equals (=)
    • Parameter 1: $("Panel2Panel3wishes1Error")
    • Source Type: Expression
    • Parameter 2: null
    • Source Type: Expression
    • Start Event: Panel2Panel3UpdatePanel.onrefresh;
  11. At Panel1 panel for UpdatePanel feature set value for attribute:
    • Refresh: Panel2Panel3UpdatePanel.onrefresh;
  12. At Panel2 panel for YahooPanel1 feature set values of attributes:
    • Fixed center: Yes
    • Show event: Panel1wishesLink1.onclick;
    • Hide event: Panel1Condition1.ontrue;
  13. At Panel3 panel for UpdatePanel feature set value for attribute:
    • Refresh: Panel2YahooPanel1.onhide;

Database Tables used:

Database: Internet

Tables: wishes
